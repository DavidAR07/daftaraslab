Impor Data Kelulusan Asisten

Format File CSV

File CSV harus memiliki header berikut (case-sensitive) dan urutan kolom tidak masalah:

  • `Nama`: Nama lengkap pendaftar.
  • `NRP`: Nomor Induk Mahasiswa (harus 10 digit angka).
  • `Keterangan`: Status kelulusan (`Lolos` atau `Tidak Lolos`).
  • `Password`: Kata sandi untuk login pendaftar.

Contoh isi file CSV:

Nama,NRP,Keterangan,Password
Budi Santoso,1234567890,Lolos,passbudi
Siti Aminah,0987654321,Tidak Lolos,passsiti
Joko Susilo,1122334455,Lolos,passjoko

Pilih file CSV yang valid.
