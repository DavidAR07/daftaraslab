Impor Data Kelulusan Tahap Interview

Pengaturan Global Langkah Selanjutnya

Link grup komunikasi dan dokumen panduan ini akan berlaku untuk semua pendaftar yang Lolos Tahap Interview.

Format File CSV

File CSV harus memiliki header berikut (case-sensitive) dan urutan kolom tidak masalah:

  • `Nama`: Nama lengkap pendaftar.
  • `NRP`: Nomor Induk Mahasiswa (harus 10 digit angka).
  • `Keterangan Interview`: Status kelulusan tahap interview (`Lolos Interview`, `Tidak Lolos Interview`, atau `Menunggu`).
  • `Password`: Kata sandi untuk login pendaftar.
  • `Waktu Interview` (Opsional): Tanggal dan waktu interview spesifik per pendaftar. Contoh: `2025-01-10T10:00`
  • `Breakout Room` (Opsional): Nomor atau nama breakout room spesifik per pendaftar.

Contoh isi file CSV:

Nama,NRP,Keterangan Interview,Password,Waktu Interview,Breakout Room
Budi Santoso,1234567890,Lolos Interview,passbudi,2025-01-10T10:00,Room A
Siti Aminah,0987654321,Tidak Lolos Interview,passsiti,
Joko Susilo,1122334455,Menunggu,passjoko,2025-01-11T13:00,Room B

Pilih file CSV yang valid.
