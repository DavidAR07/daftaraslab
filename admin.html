Link grup komunikasi ini akan berlaku untuk semua pendaftar yang Lolos Tahap Interview.

Format File CSV

File CSV harus memiliki header berikut (case-sensitive) dan urutan kolom tidak masalah:

  • `Nama`: Nama lengkap pendaftar.
  • `NRP`: Nomor Induk Mahasiswa (harus 10 digit angka).
  • `Keterangan Interview`: Status kelulusan tahap interview (`Lolos Interview`, `Tidak Lolos Interview`, atau `Menunggu`).
  • `Password`: Kata sandi untuk login pendaftar.
  • `Perkenalan`: Nilai (0-100) untuk Perkenalan, kelebihan dan kekurangan.
  • `Kesibukan`: Nilai (0-100) untuk Kesibukan akademik, dan non-akademik.
  • `Komitmen`: Nilai (0-100) untuk Skala prioritas/komitmen.
  • `Praktikum1`: Nilai (0-100) untuk Penjelasan judul praktikum pilihan 1.
  • `Praktikum2`: Nilai (0-100) untuk Penjelasan judul praktikum pilihan 2.
  • `LaporanDummy`: Nilai (0-100) untuk Yang diketahui mengenai format laporan Dummy POMITS.
  • `Ralat`: Nilai (0-100) untuk Yang diketahui mengenai ralat yang digunakan pada laporan POMITS.
  • `SkemaWaktu`: Nilai (0-100) untuk Skema pembagian waktu saat praktikum.
  • `KomitmenInputNilai`: Nilai (0-100) untuk Komitmen dalam ketepatan untuk menginput nilai.
  • `KeyakinanDiterima`: Nilai (0-100) untuk Seberapa yakin diterima jadi aslab.
  • `Keterangan`: Opsional, keterangan tambahan dari admin.

Contoh isi file CSV:

Nama,NRP,Keterangan Interview,Password,Perkenalan,Kesibukan,Komitmen,Praktikum1,Praktikum2,LaporanDummy,Ralat,SkemaWaktu,KomitmenInputNilai,KeyakinanDiterima,Keterangan
