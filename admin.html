Impor Data Kelulusan Asisten

Format File CSV

File CSV harus memiliki header berikut (case-sensitive) dan urutan kolom tidak masalah:

  • `NRP`: Nomor Induk Mahasiswa (harus 10 digit angka).
  • `Password`: Kata sandi untuk login.
  • `Nama`: Nama lengkap pendaftar.
  • `Kode`: Kode unik pendaftar (misal: A1, B2).
  • `Status`: Status kelulusan (`lulus` atau `tidak`).

Contoh isi file CSV:

NRP,Password,Nama,Kode,Status
5001232001,hidupjokowi,Ahmad Fauzi,A1,lulus
5001232002,hidupjokowi,Budi Santoso,A2,tidak
5001232003,hidupjokowi,Siti Nurhaliza,A3,lulus

Pilih file CSV yang valid.
