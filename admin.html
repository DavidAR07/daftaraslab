Pengaturan Global Interview

Link Zoom dan Dokumen Teknis ini akan berlaku untuk semua pendaftar yang Lolos.

Format File CSV

File CSV harus memiliki header berikut (case-sensitive) dan urutan kolom tidak masalah:

  • `Nama`: Nama lengkap pendaftar.
  • `NRP`: Nomor Induk Mahasiswa (harus 10 digit angka).
  • `Keterangan`: Status kelulusan (`Lolos` atau `Tidak Lolos`).
  • `Password`: Kata sandi untuk login pendaftar.
  • `Waktu Interview` (Opsional): Tanggal dan waktu interview spesifik per pendaftar. Format: `DD/MM/YYYY HH:MM`. Contoh: `31/12/2025 10:45`
  • `Breakout Room` (Opsional): Nomor atau nama breakout room spesifik per pendaftar.

Contoh isi file CSV:

Nama,NRP,Keterangan,Password,Waktu Interview,Breakout Room
Budi Santoso,1234567890,Lolos,passbudi,31/12/2025 10:45,Room A
Siti Aminah,0987654321,Tidak Lolos,passsiti,
Joko Susilo,1122334455,Lolos,passjoko,01/01/2026 13:00,Room B
